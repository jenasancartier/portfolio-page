Projects

  • Band Site

    A website for a band called The Screaming Yetis. This project was an exercise in utilizing the box model to add margins, borders, and padding around web page content. It was also an exercise in creating sidebars and wrapping text around images.

  • Resume Site

    A website to house a resume for a peculiar child with magical powers. This project was an exercise in applying divs and spans to style different sections of the resume, using HTML and CSS.

  • Cupcake Site

    A website for a cupcake shop called Gramma's Cups. This project was an exercise in applying classes to multiple HTML tags, and addings links to images, using HTML and CSS.

Why I chose to become a programmer

I've always been interested in technology and software development. Many of my closest friends are successful computer programmers, and their enthusiastic interest on the subject has inevitably rubbed off on me over the years. I come from an artistic family, with a strong background in fine arts, theatre, and film acting. I've spent the last five years persuing acting full time. I enjoy it very much, but the popular term "starving artist" has become a reality for me, and I feel it's time to pursue a career in tech. I'm excited about this change, and eager to implement my innate creativity, imagination, and problem solving skills into this new vocational venture.
